# Bania — Notatki Kliniczne

Repozytorium zawiera notatki kliniczne i medyczne w języku polskim.

## Zawartość

### [Zapalenie nerwu twarzowego i trójdzielnego](./zapalenie_nerwow_twarzowego_trojdzielnego.md)
Opracowanie kliniczne dotyczące uszkodzenia nerwu twarzowego (CN VII) i trójdzielnego (CN V) u mężczyzny w wieku 45 lat. Obejmuje:
- Etiologię i objawy
- Diagnostykę różnicową (w tym współistnienie obu nerwów)
- Klasyfikację uszkodzenia mechanicznego (skala Sunderlanda)
- Leczenie zachowawcze i operacyjne
- Okna czasowe dla interwencji chirurgicznych
